# Cursor Pro 自动化工具

## 项目说明

本项目基于 [@chengazhen/cursor-auto-free](https://github.com/chengazhen/cursor-auto-free) 开源项目进行二次开发，在此特别感谢原作者的无私分享和贡献！

## 功能改进

在原项目基础上，本工具主要解决了以下问题：

1. 解决部分用户首次打开就需要验证的问题
2. 解决注册完成需要重新登录的问题
3. 优化了整体使用体验
